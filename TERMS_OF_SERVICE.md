## 2. Description

ValoTracker is a free, open-source desktop application that reads real-time match data from VALORANT's local client API and displays it in a terminal or windowed interface. No account login, no external account linkage, and no data transmission to any server operated by the author.

## 7. Auto-Updater

ValoTracker may silently check for and apply updates from its official [GitHub Releases](https://github.com/Londopy/ValoTracker/releases) page. Updates are downloaded directly from GitHub and verified by file replacement. You can opt out at any time by setting `check_updates = false` in your `config.toml`.

## 8. Discord Rich Presence

If enabled, ValoTracker communicates with the Discord desktop application on your local machine via the Discord IPC socket. No data is sent to any server operated by the author. Discord's own [Terms of Service](https://discord.com/terms) and [Privacy Policy](https://discord.com/privacy) apply to that integration.

## 9. Privacy

ValoTracker does **not** collect, transmit, or store any personal data on external servers. All data fetched from VALORANT's local API stays on your machine. The only network requests made by ValoTracker are:

- The GitHub Releases API (version check, once per 24 hours, opt-out available).
- The GitHub asset download URL (only when a newer version is found).

No analytics, telemetry, or crash reporting is included.
